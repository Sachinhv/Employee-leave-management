## Employee Leave Management System

## 📌 Project Purpose
This project is built to manage employee leave requests in an organization.
It allows employees to apply for leave and managers to approve or reject requests.

## 🚀 Features
- User Authentication (Login/Register)
- Role-based Access (Employee / Manager / Admin)
- Apply for Leave
- Leave Approval System
- Dashboard with Leave Statistics
- Email Notifications

## 🛠 Tech Stack
- Django
- PostgreSQL / SQLite
- Bootstrap
- Vercel Deployment
